Auto Whip Switching is a quality-of-life mod made for summoners to streamline whip combat.

🧠 Key Features:

Automatically switches to sub-whip when corresponding buff is missing

Switches back to main whip when all buffs are present

Supports main whip rotation: cycle through selected whips (ordered by damage)

Optionally switch early when buff duration is about to expire

Fully localized in English and Chinese

🛠️ How to Use:

Open Mod Config:

Define your main whips: choose one or more for rotation when buffs are sufficient

Define sub-whips and their corresponding buffs

Set rotation duration (in frames) and buff threshold (frames)

Optionally enable log messages to see when whips are switching

Use the Toggle Key:

Bind a hotkey for toggling the auto-switch feature on/off

🔔 Switches only happen while actively attacking—no accidental changes.

💡 This is a client-side mod. No server installation needed.

Master your whips with effortless automation!

自动切鞭 是专为召唤师设计的自动化辅助模组，助你释放双手、专注战斗。

🧠 模组功能简介：

玩家未获得副鞭 Buff 时，自动切换至对应副鞭

玩家 Buff 齐全后自动切回主鞭

主鞭支持轮换系统（根据伤害排序，每条鞭子使用固定帧数后轮换）

可在 Buff 即将消失前提前切换副鞭（可选）

支持完整的中英文本地化

🛠️ 模组使用指南：

打开设置（模组配置）：

设置主鞭：点击添加按钮，选择一或多条你想在 Buff 齐全时使用的鞭子

设置副鞭与 Buff：为每条副鞭设置对应的 Buff（系统会根据缺失 Buff 自动切换）

设置主鞭轮换时间（单位：帧）和 Buff 检测阈值（单位：帧）

启用日志提示（可选）帮助你调试或观察切鞭过程

快捷键控制：

在设置界面绑定「切鞭开关」快捷键，按下可启用/禁用自动切鞭

🔔 所有切鞭仅在攻击时发生，避免非战斗状态误触。

💡 本模组完全客户端运行，无需服务端安装。

解放双手，尽享召唤！
