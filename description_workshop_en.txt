[h1]Auto Whipstacking Mod[/h1]

A smart auto-switching mod for Summoners and buff-dependent players. Stay focused on movement while your weapons are switched automatically based on buff status, debuff cooldowns, and rotation setup.

[h2]🆕 New Features (v2.3.0)[/h2]
[list]
[*] ✅ Added [b]independent toggles[/b] for [b]main weapon / sub weapon / debuff weapon[/b], allowing you to enable or disable each switching system separately in the config UI
[*] ✅ [b]Redesigned configuration UI[/b] with a cleaner layout, improved visual clarity, and better user experience
[*] ✅ Optimized core switching logic for better stability and fixed some cases where weapons failed to switch back properly
[/list]

[h2]✨ Core Features[/h2]
[list]
[*] Automatically switches to sub weapon when buffs are missing
[*] Switches back to main weapon when buffs are present
[*] Supports multiple main weapons with shared usage duration
[*] GUI config interface for all weapon roles and buff bindings
[*] Multilingual support and optional log output
[/list]

[h2]🛠 How to Use[/h2]
[list]
[*] Open mod config and set:
   - Main weapons for rotation
   - Sub weapons and their corresponding buffs
   - Debuff weapons and their interval (in seconds)
[*] You can bind a toggle key in Controls to turn auto-switching on/off
[*] The mod will handle weapon switching automatically during combat
[/list]
